# @amlplugins/google-youtube

Native `@googleapis/youtube` SDK re-exported under the `@amlplugins` namespace.

> YouTube Data (youtube_v3).

## Install

```bash
npm install @amlplugins/google-youtube
```

## Use

```ts
import { /* named exports from @googleapis/youtube */ } from "@amlplugins/google-youtube";
// or
import sdk from "@amlplugins/google-youtube";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
